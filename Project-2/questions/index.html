  1. What are the main differences between external, internal, and inline CSS?

    • External CSS has a separate file which is linked in the head of HTML document.
    • Internal CSS is declared inside <style></style> element, in the head of HTML document.
    • Inline CSS is declared inside an element as an attribute where style is the attribute and properties plus their values are the attribute value.

  2. What is the syntax for class and ID selectors?

    • Syntax for class selectors: .class
    • Syntax for ID selectors: #idName

  3. How would you apply a single rule to two different selectors?

    4. selector1, selector2 {property:value;}

  4. Given an element that has an id of title and a class of primary, how would you use both attributes for a single rule?

    5. #title.primary{property:value;}

  5. What does the descendant combinator do?

    6. The descendant combinator is represented by a single space between selectors and has effect on the last selector only if it's nested inside of ancestor.

  6. Between a rule that uses one class selector and a rule that uses three type selectors, which rule has the higher specificity?

    7. The rule which includes the class.

  7. From inside to outside, what is the order of box-model properties?

    1. Content
    2. Padding
    3. Border
    4. Margin

  8. What does the box-sizing CSS property do?

    9. Box-sizing property sets how the total width and height of an element is calculated. It can have the value content-box which gives the default CSS box-sizing behavior or it can have the value border-box which makes the value of the border and padding to shrink to the element width and height.

  9. What is the difference between the standard and alternative box model?

    10. The standard box model is represented by the given dimensions to the content box and if there will be specified a size for padding or border, that size will be added to the size of the content box.
    The alternative box model is represented by the box that is seen on the screen, this means that the size of the padding and border will be included in the size of the content box.

  10. Would you use margin or padding to create more space between 2 elements?

    11. I would use margins as this is placed in the extremity of the box.

  11. Would you use margin or padding to create more space between the contents of an element and its border?

    12. I would use padding because this property is placed between content box and border.

  12. Would you use margin or padding if you wanted two elements to overlap each other?

    13. I would use margin.

  13. What is the difference between a block element and an inline element?

    14. A block element is placed on a single line, below and above other block element.
    An inline element can be places beside another element.

  14. What is the difference between an inline element and an inline-block element?

    15. Inline elements have width and height ignored.
    Inline-block elements behave like an inline element with width, height, margins and paddings are respected.

  15. Is an h1 block or inline?

    16. Is a block element.

  16. Is button block or inline?

    17. Is an inline element.

  17. Is div block or inline?

    18. Is a block element.

  18. Is span block or inline?

    19. Is an inline element.

  19. What’s the difference between a flex container and a flex item?

    20. A flex container is an element that has the property display:flex.
    A flex item is any element located inside the flex container.

  20. How do you create a flex item?

    21. A flex item can be created in the same way as the flex container, that means it can be both flex item or flex container.

  21. What are the 3 values defined in the shorthand flex property?

    1. flex-grow
    2. flex-shrink
    3. flex-basis

  22. How do you make flex items arrange themselves vertically instead of horizontally?

    23. Assigning the value to be column on flex-direction property.

  23. What is the difference between justify-content and align-items?

    24. justify-content is applied for MAIN AXIS.
    align-items is applied for CROSS AXIS.

  24. How do you use flexbox to completely center a div inside a flex container?

    25. By defining the following properties to the container: justify-content:center and align-items: center.

  25. What’s the difference between justify-content: space-between and justify-content: space-around?

    26. justify-content: space-between is taking into account only the space between elements, not the space between the extremities.
    justify-content: space-around is assigning an equal space between the elements and also between extremities.
